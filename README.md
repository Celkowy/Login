# Login

Notes creator with registration and login for existing users.


Goal is to:
```
- Create an account
- Sign in
- Create title
- Write note
- Save
- Logout
```




```
v1.0
- Registration
- Login in
- Notes without saving
- Logout
```

```
v1.1
- Switching between login and register activity
```

```
v1.2
- Exit from application
```

```
v2.0
Fingerprint authentication
```

# I used

- Java
- Firebase
